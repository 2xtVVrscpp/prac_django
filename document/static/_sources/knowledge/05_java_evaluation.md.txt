# 式の評価

## 演算の優先順位
プログラムにおいて演算の優先順位はカッコ()が最優先、ついで、**積**/**商**/**余り**、最後に**和**/**差**
となります。
また、演算は左から右に行われます。

```
num = 1+3*5+(2+6)
1. 2+6 = 8
2. 3*5 = 15
3. 1+15 = 16
4. 16+8 = 24
5. num = 24 
```


## 型の変換
ここで注意となるのは演算をしたときに
小数点以下の数値が切り捨てられる場合です。
どういったときに起こるのかというと``整数型/整数型``の時です。
よくわからないかもしれないのでソースコードの例を見てください。
詳しくはキャストで調べてみてください。

```
(例)
int num1=3, num=2;
double num3=3.0;

num1/num2 ← 小数点は切り捨てられる
num3/num2 ← 小数点は切り捨てられない
```
