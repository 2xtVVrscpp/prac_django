# メモリの解放

## ガベージコレクション GC(Gabage Collection)
プログラムを実行する際に、intなどで変数指定するたびに
パソコンのメモリが消費されていきます。
何もしない場合、使われていない変数によっても
メモリが圧迫されてしまいます。
そこで定期的に、参照されなくなった変数のメモリを
自動で解放する機能がガレージコレクションです。


## 配列のメモリ開放
用途のなくなった配列に使われているメモリをガベージコレクションに
開放してもらいましょう。

```
int[] array = new int(5);

// arrayの参照先を抹消する。虚空を参照するように
array = null;
```
